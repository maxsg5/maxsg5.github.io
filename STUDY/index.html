1. Inline vs Block Elements

  • Inline Elements: e.g., <span>, <a>
  • Block Elements: e.g., <div>, <p>

2. Semantic Tags

Tags like <header>, <nav>, <main> that convey meaning.

3. HTML Indenting and Nesting Structure

<div>
  <p>This is a nested element.</p>
</div>

4. Normalize

A CSS file that ensures consistency across browsers.

5. CSS Selectors

  • p: selects all <p> elements.
  • .className: for class="className".
  • #idName: for id="idName".

6. Specificity and Combinators

  • div p: selects all <p> inside a <div>.
  • div > p: direct children only.

7. Child Element Selectors

Selects direct children. E.g., div > p.

8. Pseudo-classes

Define special states like :hover.

9. Techniques for Centering Content

  • Horizontal: margin: auto with width.
  • Vertical: Flexbox or CSS Grid.

10. Classes and IDs

  • Class: reusable styling.
  • ID: unique styling or scripting.

11. Bootstrap Layout Classes

  • .container or .container-fluid for wrapping content.
  • .row for horizontal groups.
  • .col-* for columns.
